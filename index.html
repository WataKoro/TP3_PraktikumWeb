Nvidia Corporation

Nvidia Corporation is an American multinational technology company incorporated in Delaware and based in Santa Clara, California.

It is a fabless company which designs graphics processing units (GPUs) for the gaming and professional markets, as well as system on a chip units (SoCs) for the mobile computing and automotive market.

In addition to GPU manufacturing, Nvidia provides an application programming interface (API) called CUDA that allows the creation of massively parallel programs which utilize GPUs.
